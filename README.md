# Card3d

Create beautilful 3d card effect with only attributes (idea inspired form vanilla tilt js)

## Update

[Changelog](./CHANGELOG.md)

## Installation

```
$ npm i card3d
```

or from the cdn

```html
<script src="https://unpkg.com/card3d@2.6.4/dist/card3d.js"></script>
```
